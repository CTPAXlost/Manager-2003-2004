# Футбольный менеджер 2003 — v1.1.1

## Исправление GitHub Actions

- Исправлено падение проверки проекта из-за повреждения кириллических имён файлов при распаковке ZIP и загрузке через браузер GitHub.
- Исходники установщика теперь имеют безопасные ASCII-имена: `INSTALL_GAME.cmd` и `INSTALL_GAME.ps1`.
- Во время сборки GitHub Actions копирует их в Windows-архив уже под русскими именами.
- Валидатор больше не зависит от сохранения кириллицы в имени файла репозитория.
- Игровая логика и база v1.1.0 не изменялись.
