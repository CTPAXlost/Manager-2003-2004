# Изменения версии 0.2.0

## Состав и тактика

- Удалено дублирование одного футболиста сразу на нескольких позициях.
- При загрузке старой карьеры стартовый состав автоматически очищается от дублей.
- Карточки футболистов снова можно перетаскивать на схему.
- Добавлен надёжный запасной вариант: сначала нажать на футболиста, затем на позицию.
- Футболисты на поле меняются местами, а игрок со скамейки до матча заменяет выбранного игрока основы.
- При смене схемы игра старается сохранить выбранных футболистов и распределяет их без повторений.

## Замены

- В матче разрешено пять замен.
- Окно замен показывает каждого игрока основы только один раз.
- Снятый футболист попадает в список `subbed_out` и больше не может вернуться на поле.
- Уже выпущенного футболиста нельзя выбрать повторно.
- Во время матча игрок со скамейки выходит только через кнопку «Замена».

## Симуляция матча

- Каждый отрезок теперь может содержать несколько атак.
- Вероятность момента и вероятность гола рассчитываются отдельно.
- Сила состава по-прежнему даёт 60% результата, случайность — 30%, удача — 10%.
- Для затянувшегося матча 0:0 немного повышается интенсивность концовки.
- Автоматический тест на 30 000 матчах даёт около 2,5 гола за игру и примерно 7% результатов 0:0.

## Старые сохранения

При загрузке старой карьеры игра берёт актуальную встроенную базу и накладывает на неё прогресс сохранения. Новые футболисты добавятся в составы автоматически. Для чистейшей проверки механики всё же лучше начать новую карьеру.
