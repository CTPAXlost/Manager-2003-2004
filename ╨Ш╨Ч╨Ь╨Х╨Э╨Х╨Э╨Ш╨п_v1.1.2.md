# Изменения v1.1.2

- Убраны карикатурные круглые лица: на поле используются чистые стилизованные фигуры футболистов в форме.
- Исправлено управление составом: карточка перетаскивается с меньшего движения, ScrollContainer больше не перехватывает жест; добавлен надёжный режим «нажал игрока → нажал позицию» с заметной подсветкой.
- Исправлена статистика соревнований: отдельные данные по каждой лиге, скрыты нулевые строки, добавлено восстановление разреженной статистики старых сохранений.
- Перебалансирована сила команд: разница рейтингов сильнее влияет на результат, случайность не перекрывает большую разницу класса.
- Усталость снижена до реалистичного темпа, восстановление между матчами усилено; старые сохранения с массовой формой 35–45% автоматически исправляются.
