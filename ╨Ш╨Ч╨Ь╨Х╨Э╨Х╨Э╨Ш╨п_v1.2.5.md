# Изменения v1.2.5

## Что сделано
- Для **Реал Мадрид** добавлены новые **портретные карточки игроков** в тактической схеме.
- Карточки сделаны в отдельном визуальном стиле: тёмно-синий фон, золотая рамка, компактный портрет, более узнаваемые лица.
- Новые карточки подключены только для ключевых игроков Реала:
  - Iker Casillas
  - Roberto Carlos
  - Iván Helguera
  - Francisco Pavón
  - Míchel Salgado
  - David Beckham
  - Zinedine Zidane
  - Esteban Cambiasso
  - Luís Figo
  - Santiago Solari
  - Raúl
  - Ronaldo
- Для остальных клубов и игроков оставлен текущий упрощённый режим отображения.
- Размер блока портрета в карточке на поле увеличен, чтобы мини-карточки читались лучше.

## Назначение версии
Эта сборка нужна именно для проверки, подходит ли такой формат карточек тебе визуально. Если вариант зайдёт, следующий шаг — перенос такого же подхода на другие команды и/или на всю лигу.
