# Изменения v0.3.0

## Интерфейс

- Убраны чёрные поля и жёсткое смещение интерфейса при разворачивании окна.
- Добавлен режим `expand` и минимальный размер окна 1024×680.
- Основная рабочая область теперь прокручивается, поэтому панели не обрезаются.
- Экран тактики становится вертикальным на небольших разрешениях.
- Минимальная ширина футбольного поля уменьшена с 720 до 620 пикселей.

## Стартовый состав

- Для каждой из 11 позиций появился отдельный выпадающий список всех игроков клуба.
- Если выбран футболист, уже занявший другую позицию, игроки меняются местами.
- Перетаскивание по полю и выбор «игрок → позиция» сохранены.
- Добавлена кнопка «Подтвердить состав».
- Матч нельзя начать, пока не выбраны 11 разных игроков и вратарь.
- Исправлена ошибка, из-за которой во время матча сила состава могла стать равной нулю.

## Замены

- Лимит: пять замен.
- Окно замен переписано на отдельное адаптивное окно.
- Оба списка имеют читаемый тёмный фон, крупный текст и полные имена игроков.
- Снятый игрок не может вернуться.
- Уже вышедший запасной не может выйти повторно.
- Замены больше не меняют стартовую одиннадцатку следующего матча.

## Статистика

Добавлен отдельный раздел:

- лучшие бомбардиры;
- лучшие ассистенты;
- сухие матчи вратарей;
- жёлтые карточки;
- красные карточки;
- количество сыгранных матчей.

Статистика рассчитывается для управляемой команды и для автоматически сыгранных матчей соперников, сохраняется вместе с карьерой и обнуляется в начале нового сезона.

## Симуляция

- Средняя тестовая результативность: около 2,41 гола за матч.
- Матчи 0:0: около 7,6%.
- Сохранена модель 60% сила состава / 30% случайность / 10% удача.
- Карточки теперь получают конкретные футболисты.
- Удаление временно снижает силу команды в текущем матче.
